## COMMANDS ##
**To install the  node module**

     yarn install
     
**To install Hardhat**

     npm install --save-dev hardhat

**Start the Hardhat local Server**

    npx hardhat node
    
**To deploy the solidity program**

     yarn hardhat run scripts/deploy.js

**To start the react app**

     yarn start
